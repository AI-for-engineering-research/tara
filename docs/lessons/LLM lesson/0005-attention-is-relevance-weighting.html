Attention is relevance weighting

Core idea: Attention lets a token weigh which other tokens in context matter most for interpreting or generating it.

What to remember

  • Attention is about relevance, not memorization.
  • A token can use multiple other tokens, not just one.
  • Context helps disambiguate meaning.

Why it matters

This is how transformers handle long-range dependencies without processing one token at a time like an RNN.

Mental model

For a token, compute how relevant other tokens are, then mix information from them accordingly.

Checkpoint
