Training vs inference

Core idea: Training changes weights; inference uses fixed weights.

What to remember

  • Training uses real text to provide the next-token targets.
  • Inference does not update weights.
  • Chat adaptation usually comes from context, not live learning.

Why it matters

This distinction explains why the model is not normally learning permanently during a conversation.

Mental model

Training: predict → compare to actual next token → update weights. Inference: predict → choose token → continue.

Checkpoint

  1. How does the model know the “correct” next token during training?
  2. What changes during training?
  3. What stays fixed during inference?
