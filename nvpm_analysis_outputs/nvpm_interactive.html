Functions considered

All functions first model g(H,F) = relative nvPM EIn versus the 13.8% hydrogen baseline, with x = H − 13.8 and F = Thrust / 100. Experimental fuel/reference ratios are then predicted as g(Hfuel,F) / g(Href,F).

ICAO-family exponential

g(H,F) = exp[(αF + β)(H − 13.8)]

Simple ICAO-style baseline correction. Thrust changes the hydrogen sensitivity.

Fitted parameters
  • alpha: 0.6135
  • beta: -0.9673

Mechanistic ICAO ratio

g(H,F) = exp[(aF² + bF + c)(H − 13.8)]

Same baseline-ratio idea, but allows a curved/nonlinear thrust dependence.

Fitted parameters
  • a_F2: -0.4425
  • b_F: 1.127
  • c: -1.076

Power-law exponential

g(H,F) = exp[(aF + b) · sign(x)|x|ᵖ]
x = H − 13.8

Lets hydrogen response be nonlinear relative to the 13.8% baseline while preserving whether H is above or below baseline.

Fitted parameters
  • a: 0.6236
  • b: -0.9819
  • p_raw: -0.7482

Saturating exponential

g(H,F) = exp{-A(F)[1 − exp(−k(H − 13.8))]}
A(F) > 0, k > 0

Allows the nvPM reduction relative to 13.8% hydrogen to level off rather than changing indefinitely.

Fitted parameters
  • a: 5724
  • b: -3631
  • k_raw: -8.686

Log-polynomial

log[g(H,F)] = c₁x + c₂x² + c₃xF + c₄x²F + c₅xF²
x = H − 13.8; g(13.8,F)=1

Flexible empirical surface anchored to the 13.8% baseline.

Fitted parameters
  • c1: -1.171
  • c2: 0.1054
  • c3: 1.422
  • c4: -0.3264
  • c5: -0.4442

Quadratic rational

g(H,F) = (1 − Ĥ)[(a + bF)Ĥ + 1]
Ĥ = (H − 13.8) / (Hinf − 13.8)

Shape-inspired hydrogen response with a finite high-hydrogen scale.

Fitted parameters
  • a: -1.045
  • b: 1.393
  • H_inf: 16.36

Quadratic rational constrained

g(H,F) = (1 − Ĥ)[(a + bF)Ĥ + 1]
Ĥ = (H − 13.8) / (Hinf − 13.8)

Same equation as quadratic rational, but fitted with physical-shape guardrails. Hover for constraints and fitted parameters.

Constraints
  • Scale constraint: Hinf must be larger than the largest observed hydrogen content and greater than 13.8%.
  • Monotonicity constraint: predicted baseline-relative nvPM EIn must not increase as H increases over the observed data range.
  • Thrust-grid check: monotonicity is checked across representative thrust values from F/F00 = 0.05 to 1.00.
Fitted parameters
  • a: -1.045
  • b: 1.393
  • H_inf: 16.36

How the 95% CI was computed: for each model, fit once to the observed fuel/reference ratios using predictions g(Hfuel,F)/g(Href,F), compute residuals (observed − fitted), repeatedly resample those residuals with replacement, add them back to the fitted predictions to make synthetic datasets, refit the same model to each synthetic dataset, and predict the baseline-relative hydrogen-response curve g(H,F). At each hydrogen value, the band is the 2.5th to 97.5th percentile of those bootstrap predictions. The plotted curves are relative to the 13.8% hydrogen baseline and show several thrust levels.
